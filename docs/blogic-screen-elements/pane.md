# Pane

This is a **group element**. It defines the beginning of a group of elements and requires a matching group end element.

A pane must be placed directly inside [`Panes`](./panes.md).

## Configuration

| Setting | Description          |
|:--------|:---------------------|
| title   | The title of the tab |
